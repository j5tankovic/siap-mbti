## Instalacija biblioteka
* Nltk  
  ```pip install -U nltk```
* Numpy  
  ```pip install numpy```
* SciPy  
  ```pip install scipy```
* Scikit-learn  
  ```pip install -U scikit-learn```
* Pandas  
  ```pip install pandas```
* Matplotlib  
  ```pip install matplotlib```
* Seaborn  
  ```pip install seaborn```
